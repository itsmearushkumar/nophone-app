# 📵 NoPhone – AI-Based Smart Phone Detection System

NoPhone is a modern Flutter + IoT based smart monitoring system designed to detect phone usage in restricted environments such as classrooms, libraries, study halls, examination centers, and workplaces.

The system combines IoT hardware sensors with a beautiful Flutter application to provide real-time alerts, monitoring, analytics, and distraction prevention.

---

# 🚀 Features

## 📱 Flutter Mobile/Web App
- Modern UI/UX with neon cyber theme
- Real-time phone detection alerts
- Animated warning system
- Circular countdown timer
- Usage analytics dashboard
- Responsive Flutter Web support
- Smooth navigation and transitions

---

## 🔌 IoT Integration
- Arduino/ESP WiFi-based communication
- Real-time sensor data transfer
- Wireless monitoring using WiFi
- Future-ready backend architecture
- Easy hardware integration support

---

# 🧠 Project Workflow

1. Sensor detects phone usage
2. Arduino/ESP processes detection
3. Data sent through WiFi
4. Flutter app receives alert
5. Warning screen activates
6. Analytics are updated in real-time

# ⚡ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/itsmearushkumar/nophone-app.git
```

---

## 2️⃣ Open Project

```bash
cd nophone-app
```

---

## 3️⃣ Install Dependencies

```bash
flutter pub get
```

---

## 4️⃣ Run Flutter App

```bash
flutter run
```
